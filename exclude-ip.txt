# 以下のIPは「スキャン」と判定したアクセス元で、集計・ログから除外
# 判定は次の3種(A〜C)のいずれかに該当（しきい値は env.ps1 で調整可）
# 共通: 総リクエスト数が一定数（既定30）未満、または
#     最も混雑した窓（既定1分）の1分あたり要求数（ピークレート）が一定（既定10件/分）未満のIPは対象外
# (A) 列挙スキャン: 異なるURLパスが一定数（既定20）以上、かつ
#     404が一定数（既定20）以上、または200応答が全体の一定割合（既定1割）未満
# (B) 高エラー型: 401/403/404/429（4xx）に5xxを重み（既定0.5）で加味したエラー量が一定数（既定30）以上、かつ
#     その値が総応答の一定割合（既定5割）以上
# (C) クレデンシャル総当たり: 単一パスへのアクセスが一定数（既定50）以上、かつ
#     401+403+429 が一定数（既定20）以上、または一定数（既定50）以上アクセスされたパスのいずれかが認証系パスに一致

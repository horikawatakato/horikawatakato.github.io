アプリを使う

主な機能

  • 現在の日付と入力日付の差分を日数で表示（365日以上は年数も併記）
  • 紀元前999年〜西暦9999年に対応（紀元前の年は負数で入力）
  • 曜日の表示

主な使用技術

  • AWS ECS on EC2でNginxをリバースプロキシに使用したコンテナ構成を構築
  • GitHub Actionsでテスト→ビルド→プッシュ→デプロイを自動化したCI/CDパイプラインを構築

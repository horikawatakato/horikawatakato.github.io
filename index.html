datecalc — 日数計算機

日数計算を行うWebアプリケーション

アプリを使う

主な機能

  • 現在の日付と入力日付の差分を日数で表示（365日以上は年数も併記）
  • 紀元前999年〜西暦9999年に対応（紀元前は負数で入力）
  • 曜日の表示

主な使用技術

  • AWS ECS on EC2で、Nginxをリバースプロキシに使用したコンテナ構成を構築
  • GitHub Actionsで、テスト→ビルド→プッシュ→デプロイを自動化したCI/CDパイプラインを構築
AWS ECS on EC2 GitHub Actions CI/CD GitHub Container Registry Docker Nginx Gunicorn Flask Python
